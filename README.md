## 📅 Day 1 - Marks Predictor

### 📌 Project

A simple Linear Regression model that predicts a student's marks based on the number of hours studied.

### 🛠 Technologies Used

- Python
- Pandas
- Scikit-Learn

### 📚 Concepts Learned

- Machine Learning Workflow
- Features (X) and Target (y)
- Pandas DataFrame vs Series
- Train-Test Split
- `test_size`
- `random_state`
- Linear Regression
- Model Training using `fit()`
- Making Predictions using `predict()`

### 📈 Dataset

| Hours | Marks |
|-------:|------:|
|1|20|
|2|25|
|3|30|
|4|35|
|5|40|

### 🚀 Prediction Example

```python
prediction = model.predict([[6]])

print(prediction)
```

Output

```
[45.]
```
